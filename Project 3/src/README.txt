Oδηγίες για την χρήση των προγραμμάτων

Στον φάκελο περιέχονται 3 υλοποιήσεις του αλγορίθμου,Makefile,oι είσοδοι σε .bin και .mat μορφή και τα αρχεία matlab.

Yλοποιήσεις:

Σεριακός 	serial_meanshift.c
CUDA shared	cuda_meanshift_sm.cu
CUDA global	cuda_global_gm.cu

(Περιλαμβάνεται και η cuda_sm_dynamic.cu για εποπτικούς λόγους χωρίς να συμμετέχει στις μετρήσεις.) 
-------------------------------------------
Aρχεία εισόδου:
 
R15 	inputr15.bin    r15.mat
A1	input3000.bin	test3000.mat
A3	input7500.bin	test7500.mat
Yeast	input8.bin	test8.mat
Dim32	input32.bin	test32.mat
Dim64	input64.bin	test64.mat
Dim128	input128.bin	test128.mat
Dim256	input256.bin	test256.mat
-------------------------------------------
Τest scripts:

test_meanshift.m
static_test.m
-------------------------------------------


Οι υλοποιήσεις δέχονται ως είσοδο τα .bin αρχεία που είναι τα δεδομένα σε double μορφή και αποθηκευμένα σε transpose μορφή έτσι ώστε να αποθηκεύονται διαδοχικά στη μνήμη οι συντεταγμένες του 1ου σημείου μετά του 2ου κλπ. Το serial_meanshift δέχεται ως ορίσματα την .bin είσοδο, το h, το .bin που θέλουμε να αποθηκευθεί η έξοδος, τον αριθμό των διαστάσεων του σετ. Π.χ. ./serial_meanshift input7500.bin 0.6 out.bin 2 
Oι υλοποιήσεις σε  CUDA δέχονται ένα επιπλέον όρισμα που είναι οι επαναλήψεις του αλγορίθμου που θέλουμε να γίνουν. Μπαίνουν ως 3ο όρισμα. Έτσι αν στο προηγούμενο αρχείο θέλουμε 25 επαναλήψεις τότε ./cuda_meanshift_sm input7500.bin 0.6 25 out.bin 2
ΠΡΟΣΟΧΗ: Η τιμή του h είναι η τετραγωνική ρίζα της τιμής που θα δίναμε στο MATLAB.
Η μεταβλητή testMode των εκτελέσιμων να έχει την τιμή 0 όταν δεν τσεκάρουμε τον αλγόριθμο μέσω ΜATLAB. 
Στους κώδικες υπάρχουν σε comments κομμάτια κώδικα για να βλέπουμε την είσοδο και την έξοδο στο terminal αν χρειαστεί.
Πάντα τα αρχεία να είναι στον ίδιο φάκελο. 

Για να κάνουμε δοκιμές στο ΜATLAB τρέχουμε το test_meanshift.m ή το static_test.m. Tα υπόλοιπα .m αρχεία πρέπει να είναι στον ίδιο φάκελο καθώς και οι εκτελέσιμοι κώδικες. Στην αρχή των scripts υπάρχει μια ενότητα %Define parametrs για τον καθορισμό των απαραίτητων παραμέτρων, με σχόλια για το τι συμβολίζει η κάθε μία. Για τα αρχεία οι ονομασίες τους φαίνονται στον πάνω πίνακα. Αναμένουμε τα αποτελέσματα στην κονσόλα του ΜΑΤLAB. Σε αυτή την περίπτωση η μεταβλητή test mode των εκτελέσιμων να έχει την τιμή 1. 

**Για λόγους συμβατότητας πρέπει να χρησιμοποιηθεί έκδοση Matlab R2017a ή νεότερη.**
