Περιεχόμενα και Oδηγίες για την χρήση των προγραμμάτων.

Στον φάκελο code περιέχονται 4 υλοποιήσεις του αλγορίθμου,Makefile,,τα χαρακτηριστικά της CPU και GPU όπου έτρεξαν οι κώδικες και 1 script για την εκτέλεση των υλοποιήσεων για όλες τις εικόνες.
Στον φάκελο test images: oι είσοδοι(original & noisy) σε .bin καi .jpeg μορφή,αποτελέσματα denoising σε .jpeg μορφή.
Στον φάκελο robustness: τα αρχεία matlab για έλεγχο ορθότητας.

Yλοποιήσεις:

Σεριακός nlm 			serialnlm.c
Σειριακός adaptive nlm		serialnlm_adaptive.c
CUDA nlm			nlm.cu
CUDA adaptive nlm		adaptive.cu
----------------------------------------------------------------------
Χαρακτηριστικά CPU, GPU: 

hostQuerry.txt
deviceQuerry.txt
----------------------------------------------------------------------
Bash Script για εκτέλεση υλοποιήσεων:

cuda_script2.sh

Matlab script για ορθότητα:
test_non_local_means.m

Πρέπει να είναι στον ίδιο φάκελο με τα αρχεία:
demo_non_local_means.m 
nonLocalMeans.m
adaptNonLocalMeans.m
----------------------------------------------------------------------


Οι υλοποιήσεις δέχονται ως είσοδο τα .bin αρχεία που είναι τα δεδομένα σε float μορφή και αποθηκευμένα σε transpose μορφή έτσι ώστε να αποθηκεύονται στη μνήμη κατά γραμμές (διότι η matlab τα επεξεργάζεται κατά στήλες).Όλες οι υλοποιήσεις δέχονται τα εξής 4 ορίσματα: την noisy εικόνα σε .bin μορφή, ενα .bin αρχείο στο οποίο θα αποθηκευτεί η denoised εικόνα, τις γραμμές της εικόνας, τις στήλες της εικόνας. Έτσι για παράδειγμα αν θέλουμε να τρέξουμε την υλοποίηση CUDA nlm για την image2 (360x241) θα κάνουμε: ./nlm in2_noisy.bin out2_nlm.bin 241 360  
Στους κώδικες υπάρχουν σε comments κομμάτια κώδικα για να βλέπουμε την είσοδο και την έξοδο στο terminal αν χρειαστεί.
Πάντα τα αρχεία να είναι στον ίδιο φάκελο. 

Για να κάνουμε δοκιμές στο ΜATLAB τρέχουμε το test_non_local_means.m. Tα υπόλοιπα .m αρχεία πρέπει να είναι στον ίδιο φάκελο καθώς και οι εκτελέσιμοι κώδικες. Στην αρχή των scripts υπάρχει μια ενότητα %Define parametres για τον καθορισμό των απαραίτητων παραμέτρων, με σχόλια για το τι συμβολίζει η κάθε μία. Για τα αρχεία οι ονομασίες τους φαίνονται στον πάνω πίνακα. Αναμένουμε τα αποτελέσματα στην κονσόλα του ΜΑΤLAB.
